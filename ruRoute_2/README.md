# ruRoute

**Умный split-routing для VPN серверов**

Российские сервисы идут напрямую через российский IP, весь остальной трафик — через зарубежный сервер. Клиентам ничего менять не нужно — всё происходит на стороне сервера автоматически.

---

## ⚠️ Важно — куда устанавливать

**Скрипт устанавливается ТОЛЬКО на российский сервер** — тот к которому подключаются клиенты.

```
Клиент
  ↓
✅ RU сервер ← сюда ставим ruRoute
  ↓
❌ NL сервер ← сюда НЕ нужно, он просто принимает трафик
  ↓
Интернет
```

На NL сервере ничего делать не нужно.

---

## Как это работает

```
Клиент (любой протокол)
        ↓
  Российский VPS
        ↓
   ┌────┴────┐
   ▼         ▼
Госуслуги  YouTube
Сбер       Instagram
ВКонтакте  и т.д.
   ↓         ↓
Российский  Зарубежный
   IP          VPS
```

Клиент видит тот же сервер с тем же ключом. Ключи и настройки менять не нужно.

## Установка

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/hjty-0/ruRoute/main/install.sh)
```

Скрипт сам:
- Определит какая панель установлена
- Покажет список outbound тегов и попросит выбрать нужный
- Скачает базы доменов из нескольких источников
- Настроит автоматическое обновление баз каждую ночь в 3:00
- Установит watcher который восстанавливает rules если панель перезаписала конфиг

---

## Источники баз доменов

ruRoute объединяет несколько проверенных источников:

| Источник | Описание |
|----------|----------|
| `domains-manual.txt` | Ручная база ruRoute — банки, госсервисы, вузы, API приложений |
| [itdoginfo/allow-domains](https://github.com/itdoginfo/allow-domains) | Основной community-список, 1.4k ⭐ |
| [hxehex/russia-mobile-internet-whitelist](https://github.com/hxehex/russia-mobile-internet-whitelist) | Актуальный список для мобильного интернета |
| [AntiZapret](https://antizapret.prostovpn.org) | Старейший проект, автоматические скрипты |
| antifilter.download | IP-диапазоны российских AS (CIDR) |

Плюс встроенный `geoip:ru` xray — все IP российских AS автоматически идут напрямую.

Обновление всех баз происходит автоматически каждую ночь в 3:00.

## Обновление баз вручную

```bash
bash /usr/local/ruRoute/update-lists.sh
```

---

## Удаление

```bash
bash /usr/local/ruRoute/uninstall.sh
```

## Полезные команды

```bash
# Статус watcher
systemctl status ruroute-watcher

# Логи в реальном времени
tail -f /var/log/ruroute.log

# Обновить базы вручную
bash /usr/local/ruRoute/update-lists.sh
```
